# WarriorFind

WarriorFind is a web application designed to help Wayne State University students and faculty report, search for, and recover lost and found items across campus.

Users can create posts for items they have lost or found, including images, location details, and descriptions. The platform makes it easier to connect people with their belongings through a simple browse and search system.

## Features
- Create posts for lost or found items
- Upload multiple images per post
- Tag posts as "Lost" or "Found"
- Search items by name
- Filter posts by date range
- Edit or update existing posts
- User authentication (Firebase)

## Tech Stack
- React
- Firebase (Firestore, Storage, Authentication)
- JavaScript
- CSS

## Purpose
Built to improve campus item recovery and reduce lost belongings at Wayne State University by creating a centralized, easy-to-use reporting system.
